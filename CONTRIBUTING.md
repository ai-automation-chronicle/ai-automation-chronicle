# Contributing

Thanks for your interest in AI Automation Chronicle.

## Ways to Contribute

- **Suggest a paper** - Open an issue with the paper title, authors, and why it's worth covering
- **Fix a typo or error** - Submit a PR with the correction
- **Report broken links** - Open an issue describing the problem

## Content Standards

- Posts follow a fixed structure (see `pattern.md`)
- All benchmark numbers and citations must be accurate and sourced
- No simplified analogies - editorial tone is professional and direct
- No em dashes - use hyphens instead

## Submitting a PR

1. Fork the repo
2. Make your changes in a new branch
3. Submit a pull request with a clear description of what was changed and why

## Issues

Use GitHub Issues for suggestions, corrections, or paper recommendations.
